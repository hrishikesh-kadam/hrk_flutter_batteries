A library to accumulate required flutter Widgets, Helper, Utility codes.

## Features

- Widgets
  - [`DateRangeWidget`]
  - [`ChoiceChipGroup`]
  - [`FilterChipGroup`]
  - [`LabelLinkInkWellWrap`]
  - [`LabelValueWrap`]
  - [`RadioSettingsTile`]
  - [`RadioDialog`]
  - [`DeferredWidget`]
- Helper, Utility codes
  - [`HrkDimensions`]
  - [`HrkColors`]
  - [`LocaleJsonConverter`]
  - [`LocaleListJsonConverter`]
  - [`LocaleJsonConverter2`]
  - [`LocaleListJsonConverter2`]
  - Text Helper
    - [`getTextPainterLaidout()`]
    - [`getLargestTextWidth()`]
- Extensions
  - [`HrkColorScheme`]
    - [`surfaceFixed`]
  - [`HrkDateTimeRange`]
    - [`ofMonth()`]

## Getting started

```console
flutter pub add hrk_flutter_batteries
```
